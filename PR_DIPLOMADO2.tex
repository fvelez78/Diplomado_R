\frame{
\frametitle{Naturaleza de la información en el mundo sensorial}
En la evaluación sensorial se consideran variables de diversa naturaleza, las cuales de manera general se pueden agrupar como:\\
\vspace{0,3cm}
\begin{enumerate}
	\item Cualitativas
	\begin{enumerate}[a.]
		\item Atributos no ordenados $\rightarrow$ \textcolor[rgb]{0,0,1}{Nominal} (Marca, Color)
		\item Atributos ordenados	$\rightarrow$\textcolor[rgb]{0,0,1}{Ordinal} (Nivel de agrado)
	\end{enumerate}
	\item Cuantitativas
	\begin{enumerate}[a.]
		\item Discretas		$\rightarrow$ \textcolor[rgb]{0,0,1}{Ordinal} (Frecuencia de uso)
		\item Continuas		$\rightarrow$\textcolor[rgb]{0,0,1}{Intervalo ó Razón} (Tiempos de duración )
	\end{enumerate}
\end{enumerate}
\vspace{0,3cm}
En resumen:\\
\begin{itemize}
	\item \textcolor[rgb]{0,0,1}{Nominal}: Nombres sin orden
	\item \textcolor[rgb]{0,0,1}{Ordinal}: Nombres con orden o números enteros
	\item \textcolor[rgb]{0,0,1}{Intervalo}: Medidas sin cero unico
	\item \textcolor[rgb]{0,0,1}{Razón}: Medidas con cero unico
\end{itemize}

}

\frame{
	\frametitle{¿Que es R?}
	Es un lenguaje de programación y un entorno para el cálculo estadístico y elaboración de gráficas básicas avanzadas; es una implementación de código abierto del lenguaje S (S-Plus), desarrollado por los Laboratorios Bell. Escrito inicialmente por Ross Ihaka y Robert Gentleman a mediados de los años 90.\\
	
	En R se puede realizar análisis hasta con 2 millones de registros y  mas de 250.000 variables. Es un programa amplio y flexible de análisis estadístico y gestión de información capaz de trabajar con datos procedentes de distintos formatos proporcionando, desde sencillos gráficos de distribuciones y estadísticos descriptivos, hasta análisis estadísticos complejos que permiten descubrir relaciones de dependencia e interdependencia, establecer clasificaciones de sujetos y variables, predecir comportamientos, etc.

\begin{frame}[fragile]
\frametitle{Dataframe}
Este es el elemento que con mayor frecuencia se emplea para almacenar datos estadísticos. En esencia es un arreglo bidimensional en el cual como es común se emplean
\begin{description}
	\item Filas $\rightarrow$ Registros
	\item Columnas $\rightarrow$ Variables
\end{description}
\begin{exampleblock}{Ejemplo}
	\begin{verbatim}
	ID<-c(1:10)
	GENERO<-c("H","M","M","M","H","H","M","M","H","H")
	EDAD<-c(18,21,19,20,19,21,20,19,21,18)
	IMC<-c(23.1,24.2,25.3,27.1,19.0,23.0,22.8,23.0,21.8,24.3)
	
	DT<-data.frame(ID,GENERO,EDAD,IMC)
	
	rownames(DT)<-c("Jhon","Luz","Olga","Sara","Luis",
	"Diego","Ana","Ivon","Juan","Carlos")
	\end{verbatim}
\end{exampleblock}

\end{frame}

\subsection{Inferencia Estadística}
%-----------------------------------------------------------------------------------
\frame{
	\frametitle{Inferencia Estadística}
	Es el procedimiento mediante el cual se pueden sacar conclusiones acerca de la población, partiendo de la información contenida en una muestra extraída de una población.\\
	En general hay dos tipos de inferencia estadística
	\begin{itemize}
		\item Estimación
		\item Prueba de Hipótesis
	\end{itemize}
	\pause
	Sobre que elementos se desarrolla la estimación o las pruebas de hipótesis?\\
	\begin{itemize}
		\item Medias
		\item Proporciones
		\item Varianzas
		\item Coeficientes de regresión
		\item Correlaciones
		\item ...
	\end{itemize}
	
}
%-----------------------------------------------------------------------------------
\frame{
	\frametitle{Estimación}
	Es el proceso mediante el cual intentamos determinar el valor de un parámetro de la población, a partir de la información de una muestra.\\
	\vspace{0.5cm}
	\textcolor[rgb]{1,0,0}{\emph{Estimación}}: es el valor numérico que asignamos a un parámetro.\\
	\textcolor[rgb]{1,0,0}{\emph{Estimador}}: es la formula utilizada para hacer la estimación.\\
	\begin{exampleblock}{Ejemplo}
		\[\bar{X}=\dfrac{1}{n}\sum_{i=1}^{n} x_{i} \qquad   \bar{X}=3.25\]
	\end{exampleblock}
	\pause
	\vspace{0.5cm}
	Hay dos tipos de estimaciones para cualquier parámetro, estos son:\\
	\begin{itemize}
		\item Estimación Puntual           
		\item Estimación por Intervalo     
	\end{itemize}
	
}
%-----------------------------------------------------------------------------------
\frame{
	\frametitle{Uso de la estimación por intervalo}
	Los IC se emplean mayoritariamente para variables continuas o también llamadas variables métricas.\\
	\begin{figure}[h]
		\centering
		\includegraphics[scale=0.55,keepaspectratio=true]{./IC_R.eps}
		% IC_R.eps: 0x0 pixel, 300dpi, 0.00x0.00 cm, bb=0 0 324 324
	\end{figure}
	El nivel de confianza denota la probabilidad de contener el valor verdadero.
}
%-----------------------------------------------------------------------------------
\frame{
	\frametitle{Hipótesis}
	\begin{itemize}
		\item \textcolor[rgb]{1,0,0}{Hipótesis}: Afirmación no probada acerca de un problema o situación especifica.
		\item \textcolor[rgb]{1,0,0}{Hipótesis científica}: es una proposición aceptable que ha sido formulada a través de la recolección de información y datos, aunque no esté confirmada, sirve para responder de forma alternativa a un problema con base científica.
		\item \textcolor[rgb]{1,0,0}{Hipótesis estadística}:En el contexto estadístico una Hipótesis se entiende como una afirmación acerca de un conjunto de parámetros que describen una característica de una o más poblaciones.
	\end{itemize}
	\vspace{0.6cm}
	Una \textcolor[rgb]{1,0,0}{Prueba de Hipótesis} entonces debe entenderse como un procedimiento mediante el cual se determina el grado de verdad m\'as probable de tal afirmación, en función de la información disponible en la muestra.
	
}

%-----------------------------------------------------------------------------------
\frame{
	\frametitle{Error Tipo I y II en Pruebas de Hipótesis}
	\begin{center}
		\begin{tabular}{|r|c|c|}\hline
			&ACEPTA $H_{0}$       &RECHAZA $H_{0}$\\ \hline
			$H_{0}$ Verdadera&Buena decisión&\textcolor[rgb]{1,0,0}{Error Tipo I}\\ \hline
			$H_{0}$ Falsa    &\textcolor[rgb]{1,0,0}{Error Tipo II}    &Buena decisión\\ \hline
		\end{tabular} 
	\end{center}
\pause
	En general se emplea la notación siguiente:
	\begin{itemize}
		\item $\alpha$: Error Tipo I
		\item $\beta$: Error Tipo II
	\end{itemize}
	\vspace{0.6cm}
	Toda prueba de hipótesis se diseña con el objeto de controlar el error tipo I y se deja libre el tipo II.\\
	\vspace{0.6cm}
	En el caso de los intervalos de confianza el término $\left(1-\dfrac{\alpha}{2}\right)\times 100\%$ denota precisamente el \emph{Nivel de confianza del intervalo}
}
%-----------------------------------------------------------------------------------
\frame{
	\frametitle{TIPOS DE PRUEBAS}
	Para varios casos de inferencia estadística existen fundamentalmente dos tipos de pruebas que en general se pueden describir como:
	\vspace{0.6cm}
	\begin{itemize}
		\item \textcolor[rgb]{1,0,0}{Paramétricas}: Estas hacen consideraciones acerca de la distribución de la población.
			\begin{itemize}
				\item Prueba t (una y dos medias)
				\item ANOVA
				\item Regresión
				\item $\ldots$
			\end{itemize}
		\pause
		\item \textcolor[rgb]{1,0,0}{No Paramétricas}: Generalmente no asume distribuciones para la población.
			\begin{itemize}
				\item Prueba de Wilcoxon, Rangos Signados
				\item Kruskal-Wallis
				\item Friedman, 
				\item $\ldots$
			\end{itemize}
	\end{itemize}
	\vspace{0.2cm}
	\textbf{NOTA IMPORTANTE}:En general estas pruebas tiendes a coincidir en la conclusión, sin embargo, en ocasiones se presentan discrepancias cuando los datos presentan tendencia hacia un lado de la escala de medida.
	
}

\frame{
	\frametitle{Una Media}
	La prueba paramétrica corresponde a la prueba t y la no paramétrica a la de Wilcoxon  (en su versión de una muestra).\\
	\vspace{0.4cm}
	\textbf{Hip\'otesis}:
	\begin{equation*}
	H_{0}:\mu=\mu_{0} \qquad  vs. \qquad  
	\left\{
	\begin{array}{l}
	H_{a}:\mu>\mu_{0} \\
	H_{a}:\mu \neq \mu_{0} \\
	H_{a}:\mu < \mu_{0}
	\end{array}
	\right.
	\end{equation*}
	donde $\mu_{0}$ es un valor fijo y conocido frente al cual se desea hacer la comparaci\'on.\\
	\begin{exampleblock}{En Rcmdr...}
		\textbf{Menú} $\rightarrow$ Estadísticos $\rightarrow$ Medias$\rightarrow$Test t para una muestra\\
		\vspace{0.3cm}

	\end{exampleblock}
}

	\begin{center}
		\begin{tabular}{lcl}\hline
			\textbf{PRUEBA} & \textbf{No. MUESTRAS} & \textbf{RESPUESTA}\\ \hline
			2-AFC & 2 & Más Fuerte o Débil\\

	Las cuatro primeras corresponden a \textbf{Métodos de selección forzada}, pues los participantes deben hallar una selección aún si no la pueden detectar claramente.
	
}
%-----------------------------------------------------------------------------------
\frame{
	\frametitle{Métodos de Selección Forzada}
	Estos métodos se pueden emplear con dos finalidades\\
	\vspace{0.3cm}
	\begin{enumerate}
		\item Pruebas de diferencia
		\item Pruebas de Preferencia
	\end{enumerate}
	\vspace{0,8cm}
	En general las seis pruebas de discriminación básicas son procedimientos basados en pruebas de hipótesis para proporciones las cuales para muestras pequeñas emplean la \textcolor[rgb]{0,0,1}{Distribución Binomial} a fin de establecer el resultado de la prueba.
	\begin{align*}
	X &\sim B(n,p) \qquad
	P[X=i]= \binom{n}{i}p^{i}(1-p)^{n-i}
	\end{align*}
	
}
%-----------------------------------------------------------------------------------
\frame{
	\frametitle{Modelo para pruebas de diferencia}

\frame{
	\frametitle{Estimación de $p_{d}$}
	En pruebas discriminativas el tamaño de muestra se establece a partir de fijar los valores $\alpha$, $\beta$ y $p_{d}$ por lo cual es indispensable establecer una estimación de tal valor.
	\begin{equation}
	\hat{p}_{d}=\dfrac{\hat{p}_{c}-p_{0}}{1-p_{0}}
	\end{equation}
	donde $\hat{p}_{c}$ es la proporción observada de aciertos en el panel. Con lo cual un intervalo de confianza del $95\%$ para $p_{d}$ será:
	\begin{equation}
	\hat{p}_{d}\pm 1.96\sqrt{V(\hat{p}_{d})}
	\end{equation}
	Donde $V(\hat{p}_{d})$ es la varianza de la estimación y está dada por:
	\begin{equation}
	V(\hat{p}_{d})=\dfrac{1}{(1-p_{0})^{2}}\dfrac{\hat{p}_{d}(1-\hat{p}_{d})}{N}
	\end{equation}
}
%-----------------------------------------------------------------------------------
\frame{
	\frametitle{Ejemplo...}
	En una prueba con 100 consumidores se le presenta a cada participante dos productos lácteos A y B, siendo el primero, el producto que actualmente esta en el mercado y B corresponde al mismo producto con un cambio importante de una materia prima. A cada participante se le pide probar los productos y determinar si son iguales o diferentes. Luego de la prueba se encuentra que 62 personas detectan la diferencia.
	¿Cuál es la proporción de discriminadores $\textcolor[rgb]{1,0,0}{p_{d}}$ para ese producto en la población?\\
	\vspace{0.1cm}
	Estimación puntual de aciertos: $\textcolor[rgb]{1,0,0}{p_{c}=0.62}$. Probabilidad de adivinar: $\textcolor[rgb]{1,0,0}{p_{0}=0.5}$\\
	\begin{equation*}
	\hat{p}_{d}=\dfrac{\hat{p}_{c}-p_{0}}{1-p_{0}} \qquad	\hat{p}_{d}=\dfrac{0.62-0.5}{1-0.5} \qquad \hat{p}_{d}=0.24
	\end{equation*}
%	\vspace{0.1cm}
	\begin{align*}
	V(\hat{p}_{d})&=\dfrac{1}{(1-p_{0})^{2}}\dfrac{\hat{p}_{d}(1-\hat{p}_{d})}{N} \qquad
		V(\hat{p}_{d})=\dfrac{1}{(0.5)^{2}}\dfrac{0.24(0.76)}{100} \qquad 	V(\hat{p}_{d})=0.097
	\end{align*}
	De donde\\
	\begin{equation*}
	\hat{p}_{d}\pm 1.96\sqrt{V(\hat{p}_{d})} \qquad \boxed{(0.05, 0.43)}
	\end{equation*}
}

\frame{
	\frametitle{Conceptos de Diseño Experimental}
	\begin{itemize}
		\item \textcolor[rgb]{0,0,1}{Factor}: Son variables controlables de interés para el investigador.
		\vspace{0.3cm}
		\item \textcolor[rgb]{0,0,1}{Nivel}: Cada uno de los valores que toma un factor (Alto, medio, bajo).
		\vspace{0.3cm}
		\item \textcolor[rgb]{0,0,1}{Variable respuesta}: Es lo que se mide o cuenta y es el objetivo de todo el diseño.
		\vspace{0.3cm}
		\item \textcolor[rgb]{0,0,1}{Tratamiento}: Son las combinaciones de todos los niveles de los factores.
		\vspace{0.3cm}
		\item \textcolor[rgb]{0,0,1}{Modelo}: Es la expresión matem?tica que describe te?ricamente la variable respuesta en función de los factores.
	\end{itemize}
}
%-----------------------------------------------------------------------------------
\frame{
	\frametitle{Conceptos de Diseño Experimental}
	\begin{itemize}
		\item \textcolor[rgb]{0,0,1}{Bloque}: Son variables categóricas que se introducen al diseño con el fin de organizar el material experimental en grupos más homogéneos entre si.
		\vspace{0.3cm}
		\item \textcolor[rgb]{0,0,1}{Unidad Experimental}: Es cada individuo, animal u objeto que se somete a un tratamiento.
		\vspace{0.3cm}
		\item \textcolor[rgb]{0,0,1}{Observación}: Cada valor de la variable respuesta en cada unidad experimental.
		\vspace{0.3cm}
		\item \textcolor[rgb]{0,0,1}{Unidad Observacional}: Es el elemento especifico sobre el cual se realiza la medición en la unidad experimental.
		\vspace{0.3cm}
		\item \textcolor[rgb]{0,0,1}{Replica}: Corresponde al número de unidades experimentales que se asignan a cada tratamiento.
	\end{itemize}
}

\frame{
	\frametitle{DISEÑOS DE UN FACTOR}
	Cuando se tiene un solo factor de interés el análisis estadístico tiene como principal objetivo determinar si existen diferencias significativas entre las medias de los tratamientos que para el caso son los mismos niveles del factor.\\
	Esto de expresa en la siguiente hipótesis estadística:
	\begin{equation}
	\boxed{H_{0}: \mu_{1}=\mu_{2}=\ldots=\mu_{k} \qquad vs.  \qquad H_{a}: \mu_{i}\neq \mu_{j} \qquad \forall i\neq j}
	\end{equation}
	Está hipótesis se prueba mediante la técnica de \textcolor[rgb]{0,0,1}{\textbf{Análisis de Varianza}}.\\
	\vspace{0.3cm}
	\pause
	\begin{alertblock}{IMPORTANTE}
		Notese que el Análisis de Varianza solo le permite determinar su las medias son iguales o si existe al menos un par de medias diferentes. Pero no le establece cuales pares de tratamientos son diferentes.
	\end{alertblock}
	
}
%-----------------------------------------------------------------------------------
\frame{
	\frametitle{Estructura de Datos}
	
	\begin{center}
		\renewcommand{\arraystretch}{1.2}
		\begin{tabular}{c|cccc|c}\hline
			\textbf{Tratamientos} & \multicolumn{4}{c|}{\textbf{Observaciones}}& \textbf{Medias}\\ \hline
			1        & $y_{11}$ & $y_{12}$ & $\cdots$ & $y_{1r}$&$\overline{y}_{1}$\\
			2        & $y_{21}$ & $y_{22}$ & $\cdots$ & $y_{21}$&$\overline{y}_{2}$\\
			$\vdots$   & $\vdots$ &          & $\ddots$   & $\vdots$& $\vdots$\\
			k        & $y_{k1}$ & $y_{k2}$ & $\cdots$ & $y_{kr}$&$\overline{y}_{k}$\\ \hline
		\end{tabular}
	\end{center}
	Modelo estadístico
	\begin{equation}
	\boxed{y_{ij}=\mu_{i}+\epsilon_{ij}}
	\end{equation}
	donde $i=1,\ldots,k \quad j=1,\ldots, r$ y además:
	\begin{description}
		\item [$\mu_{i}$:] Media del \emph{i-esimo} tratamiento
		\item [$\epsilon_{ij}$:] Error experimental
	\end{description}
	
}

\frame{
	\frametitle{Supuestos}
	El análisis de varianza hace los siguientes supuestos acerca del diseño, que deben ser validados para garantizar la confiabilidad de los resultados, estos son:
	\vspace{0.2cm}
	\begin{enumerate}
		\item Igualdad de varianza en los tratamientos (\emph{Homocedastisidad})\\
		\textcolor[rgb]{0,0,1}{Prueba de igualdad de varianzas de Bartlett o Levene}
		\item Independencia de las observaciones\\
		\textcolor[rgb]{0,0,1}{Diagramas de dispersión, Prueba de Durbin-Watson}
		\item Las observaciones se distribuyen de acuerdo a una normal\\
		\textcolor[rgb]{0,0,1}{Prueba de Normalidad}
	\end{enumerate}
	\pause
	\vspace{0.2cm}
	\textcolor[rgb]{1,0,0}{¿Y si los supuestos no se cumple?}\\
	\vspace{0.2cm}
	En general depende de cuales sean los supuestos no satisfechos, pero algunas formas de mejorar los resultados implica transformar la variable respuesta:
	
	
	\begin{itemize}
		\begin{columns}
			\begin{column}{3cm}
				\item $y \rightarrow \log{y}$
			\end{column}
			\begin{column}{3cm}
				\item $y \rightarrow \sqrt{y}$
			\end{column}
			\begin{column}{3cm}
				\item $y \rightarrow y^{\lambda}$
			\end{column}
		\end{columns}
	\end{itemize}

\frame{
\frametitle{DISEÑOS DE DOS FACTORES}
Para el caso de dos factores de interés el análisis estadístico tiene como principal objetivo determinar si existen diferencias significativas entre las medias de los tratamientos que corresponden a las combinaciones de los niveles de ambos factores.\\
Si $A$ y $B$ son los factores en estudio con $n$ y $m$ niveles respectivamente, entonces  el modelo correspondiente es:
\begin{equation}
\boxed{y_{ijk}=\mu+\alpha_{i}+\beta_{j}+\gamma_{ij}+\epsilon_{ijk}}
\end{equation}
donde $i=1,\ldots,n \quad j=1,\ldots, m$, $k=1,\ldots, r$.\\
\vspace{0.4cm}
Con ello la principal hipótesis estadística de interés es:
\begin{equation}
\boxed{H_{0}: \mu_{ij}=\mu_{st} \qquad vs.  \qquad H_{a}: \mu_{ij}\neq \mu_{st} \quad}
\end{equation}
donde $\mu_{ij}=\mu+\alpha_{i}+\beta_{j}+\gamma_{ij}$ se llama \emph{Media de celda}.
}

 \frame{
	\frametitle{ANÁLISIS CONJUNTO}
	Esta es una técnica que permite la evaluación de varios perfiles de un producto con el fin de establecer las mejores combinaciones de los niveles de los factores en estudio en relación a la valoración del producto.\\
	
	\pause
	\vspace{0.5cm}
	Este análisis permite:
	\begin{itemize}
		\item Identificar los factores mas importantes.
		\item Cuantificar cuales niveles son mejor valorados en cada factor.
		\item Obtener un puntaje para cada combinación (Perfil) evaluada.
		\item Clasificar los individuos participantes en grupos con valoraciones similares.
	\end{itemize}
	
}

El análisis conjoint (análisis conjunto) es una herramienta que surge de la investigación de mercados a fin de obtener él o los mejores perfiles de atributos que puedan lograr aumentar el impacto favorable en el consumidor o cliente final.
\begin{alertblock}{Postulado}

\end{alertblock}
\vspace{0.3cm}

\end{itemize}

\frame{
	\frametitle{Análisis Estadístico}
	
	En el Análisis Conjoint (AC) clásico, se emplea un análisis basado en un Modelo Lineal en el cual los factores (atributos) son la variables explicativas, las cuales se introducen como variables dummy y la variable respuesta es la \emph{Utilidad}. Para tres atributos el modelo más sencillo resulta ser:
	\begin{equation*}
	U=\beta_{0}+U_{1}+U_{2}+U_{3}+\epsilon_{i}
	\end{equation*}
	
   \pause
	donde los términos $U_{i}$ corresponde a la función de utilidad para el atributo i (\textcolor[rgb]{0,0,1}{partworths}).\\
	
	Dependiendo de la naturaleza de los atributos a trabajar, los \textcolor[rgb]{0,0,1}{partworths} $U_{i}$ pueden ser:
	\begin{itemize}
		\item Los atributos mismos cuando estos son valores en escala continua, esto es $U_{i}=\beta_{i}X_{i}$.
		\item Agregaciones de variables dummy correspondientes a los niveles del atributo cuando está en escala ordinal o nominal.
		\[U_{1}=\beta_{11}x_{11}+\beta_{12}x_{12} \qquad \text{Atributo de tres niveles} \]
	\end{itemize}
		
}

\frame{
\frametitle{DISEÑO PARA MEDIDAS REPETIDAS}
Un diseño en el cual varios individuos son medidos o evaluados varias veces en el tiempo en varios tratamientos se denomina \textcolor[rgb]{0,0,1}{\emph{estudio longitudinal}} o \emph{estudio panel}. \\
Este tipo de análisis permite establecer:
\begin{itemize}
\item Evolución o cambio de la variable respuesta en el tiempo
\item Interacción de los factores con el tiempo
\item Efecto de los tratamientos
\end{itemize}

En el caso de un factor con $s$ niveles, $t$ medidas en el tiempo, $n_{h}$ individuos en el grupo $h$, se tiene el modelo:
\begin{equation}
\boxed{y_{hij}=\mu+\alpha_{h}+\beta_{j}+(\alpha \beta)_{hj}+\pi_{i(h)}+\epsilon_{hij}}
\end{equation}
donde $i=1,\ldots,n_{h}$, $j=1, \ldots, t$ $h=1,\ldots, s$.\\

}
%-----------------------------------------------------------------------------------
\frame{
\frametitle{DISEÑO PARA MEDIDAS REPETIDAS}
Con ello las hipótesis estadísticas de interés son:\\
\vspace{0.4cm}

\textcolor[rgb]{0,0,1}{Grupo (Tratamientos)}
\begin{equation}
\boxed{H_{0}: \alpha_{h}=\alpha_{k} \qquad vs.  \qquad H_{a}: \alpha_{h}\neq \alpha_{k} \quad}
\end{equation}
\vspace{0.1cm}

\textcolor[rgb]{0,0,1}{Tiempo}
\begin{equation}
\boxed{H_{0}: \beta_{j}=\beta_{l} \qquad vs.  \qquad H_{a}: \beta_{j}\neq \beta_{l} \quad}
\end{equation}
\vspace{0.1cm}

\textcolor[rgb]{0,0,1}{Interacción Grupo $\times$ Tiempo}
\begin{equation}
\boxed{H_{0}: \mu_{h\cdot j}=\mu_{k\cdot l} \qquad vs.  \qquad H_{a}: \mu_{h\cdot j}\neq \mu_{k\cdot l} \quad}
\end{equation}

donde $\mu_{h\cdot j}=\mu+\alpha_{h}+\beta_{j}+(\alpha \beta)_{hj}$ con $h \neq k$ y $j \neq l$.\\
}
%-----------------------------------------------------------------------------------
\frame{
\frametitle{Supuestos del Modelo}
Este tipo de análisis de varianza considera varios supuestos como son:
\vspace{0.3cm}
\begin{itemize}
\item Los errores se distribuyen de acuerdo a una normal \[\epsilon_{hij} \sim N(0,\sigma_{\epsilon}^{2})\]
%\vspace{0.1cm}
\item Los individuos se consideran como un factor aleatorio cuya variabilidad se reúne en el término $\pi_{i(h)}$ el cual se asume cumple \[\pi_{i(h)} \sim N(0,\sigma_{\pi}^{2})\]
%\vspace{0.1cm}
\item Los errores y el efecto de los individuos no están correlacionados \[cov(\pi_{i(h)},\epsilon_{hij})=0\]
%\vspace{0.1cm}
\item La homogeneidad de varianzas en los tiempos de medida este modelo se denomina \textcolor[rgb]{0,0,1}{Esfericidad}.
\end{itemize}

}

	\frametitle{Naturaleza multivariada en Sensorial}
	La descripción estadística simultanea de cualquier situación mediante más de una variable corresponde en general a un análisis multivariado.\\
	En análisis sensorial esto se presenta con mucha frecuencia pues un producto se suele describir con múltiples atributos, los cuales estón típicamente correlacionados entre si.\\
	\vspace{0.3cm}
	Algunas técnicas multivariadas son:
	\begin{itemize}
		\item Análisis de componentes principales
		\item Análisis de correspondencias simples y múltiples
		\item Análisis cluster
		\item Análisis discriminante
		\item Escalamiento Multidimensional
		\item Regresión múltiple
		\item ...
	\end{itemize}
	
}
%-----------------------------------------------------------------------------------
\frame{
	\frametitle{Elementos Básicos de Análisis Multivariado}
	\begin{itemize}
		\item Vector de medias \textcolor[rgb]{0,0,1}{\texttt{M=sapply(DT,mean)}} \\
		\[\mathbf{\mu}=(\mu_{1},\mu_{2},\mu_{3})\] 
		\item Matriz de varianzas-covarianzas \textcolor[rgb]{0,0,1}{\texttt{cv=cov(DT)}} \\
		\[\mathbf{\Sigma}=\begin{pmatrix}
		\sigma_{1}^{2} & \sigma_{12} & \sigma_{13} \\
		\sigma_{12} & \sigma_{2}^{2} & \sigma_{23} \\
		\sigma_{13} & \sigma_{23} & \sigma_{3}^{2}
		\end{pmatrix}
		\]
		\item Matriz de correlaciones \textcolor[rgb]{0,0,1}{\texttt{cr=cor(DT)}} \\
		\[\mathbf{\rho}=\begin{pmatrix}
		\rho_{1} & \rho_{12} & \rho_{13} \\
		\rho_{12} & \rho_{2} & \rho_{23} \\
		\rho_{13} & \rho_{23} & \rho_{3}
		\end{pmatrix}\]
	\end{itemize}
	
}

\subsection{Análisis de Componentes Principales}
%-----------------------------------------------------------------------------------
\frame{
	\frametitle{Análisis de Componentes Principales ACP}
	El ACP es una técnica de reducción de dimensión que permite crear nuevas variables sintéticas que logren reunir la mayor cantidad de la información original (\emph{Varianza}) a fin de  lograr representaciones gráficas y análisis estadísticos más sencillos.
	
	\begin{columns}
		\begin{column}{5cm}
			\begin{figure}[h]
				\centering
				\includegraphics[scale=0.15,keepaspectratio=true]{./pca.png}
				% pca.png: 0x0 pixel, 300dpi, 0.00x0.00 cm, bb=
			\end{figure}
		\end{column}
		\begin{column}{4.5cm}
			El conjunto de variables originales tiene la caracterástica de ser correlacionadas, en tanto que las componentes principales son independientes.
		\end{column}
	\end{columns}
}
%-----------------------------------------------------------------------------------
\frame{
	\frametitle{Naturaleza de las variables}
	El Análisis de Componentes Principales (ACP) es en esencia una técnica matemática que en principio no hace ningún tipo se supuesto acerca de las variables, las cuales en general son de tipo cuantitativo, típicamente en escalas continuas.\\
	De acuerdo al interés de las variables en el ACP estas pueden ser:\\
	\begin{itemize}
		\item \textcolor[rgb]{0,0,1}{Activas}: Son las variables a partir de las cuales se construye todo al análisis y cuya dimensión se desea reducir.
		\vspace{0.3cm}
		\item \textcolor[rgb]{0,0,1}{Suplementarias}:Son variables cuantitativas o cualitativas que no aportan información en la construcción de las componentes pero sobre las cuales si hay interés de visualizar en los planos factoriales a fin de establecer asociaciones.
	\end{itemize}
	Aunque menos frecuente, algunos individuos también puede incluirse como \textcolor[rgb]{0,0,1}{Suplementarios} con una finalidad similar al caso de las variables.
}
%-----------------------------------------------------------------------------------
\frame{
	\frametitle{Construcción de las Componentes Principales}
	Las direcciones en el espacio en las cuales se presenta las mayor variación de los datos se denominan \textcolor[rgb]{0,0,1}{\emph{Componentes Principales}} y están dados por una combinación de todas las variables originales así:
	\begin{align*}
	F_{1}&=\alpha_{11}X_{1}+\alpha_{12}X_{2}+\ldots+\alpha_{1p}X_{p}\\
	F_{2}&=\alpha_{21}X_{1}+\alpha_{22}X_{2}+\ldots+\alpha_{2p}X_{p}\\
	&\vdots\\
	F_{k}&=\alpha_{k1}X_{1}+\alpha_{k2}X_{2}+\ldots+\alpha_{kp}X_{p}
	\end{align*}
	Donde los coeficientes $\alpha_{ij}$ describen la contribución porcentual de la variable $j$ en el factor $i$ y se denomina \textcolor[rgb]{0,0,1}{\emph{Cargas factoriales}} o simplemente \textcolor[rgb]{0,0,1}{\emph{Contribuciones}}.
}
%-----------------------------------------------------------------------------------
\frame{
	\frametitle{Elementos importantes del ACP}
	Tipicamente, las salidas de un ACP comprenden:
	\begin{itemize}
		\item \textcolor[rgb]{0,0,1}{Valores propios}:Brindan la información de cuanta varianza esta explicada en cada componente.
		\vspace{0.2cm}
		\item \textcolor[rgb]{0,0,1}{Contribuciones}: Describen la contribución porcentual de la variable $j$ en el factor $i$.
		\vspace{0.2cm}
		\item \textcolor[rgb]{0,0,1}{Matriz de correlaciones}: Cuantifica el grado de asociación entre las variables analizadas.
		\vspace{0.2cm}
		\item \textcolor[rgb]{0,0,1}{Coordenadas}: Tanto para variables como para individuos se puede obtener sus coordenadas en cada componente, de tal manera que se pueden representar gráficamente.
		\vspace{0.2cm}
		\item \textcolor[rgb]{0,0,1}{Gráficas factoriales}: A partir de las dos primeras componentes se pueden hacer representaciones bidimensionales.
	\end{itemize}
	% ($\cos^{2}{\theta}$)
}

\frame{
\frametitle{Análisis de Factorial Múltiple}
El AFM es una técnica de reducción de dimensión que mediante la construcción de nuevas variables denominadas factores permite obtener identificar la estructura latente de las variables.

%-----------------------------------------------------------------------------------
\frame{
\frametitle{Análisis de Factorial Múltiple}
\textbf{Objetivo del AFM}: Describir la relación de covariación entre múltiples variables en términos de pocas variables no observables llamadas \emph{Factores}.\\

El AFM puede ser de dos tipos dependiendo la finalidad con la cual se realice:

\begin{itemize}
\item \textcolor[rgb]{0,0,1}{Exploratorio}: En este caso no se tiene información acerca de la estructura del problema y se quiere precisamente establecer que variables latentes (factores) están presentes.
\vspace{0.3cm}
\item \textcolor[rgb]{0,0,1}{Confirmatorio}: Bajo una estructura o agrupamiento de variables manifiestas en factores, el AFM provee una confirmación o negación de la misma.
\end{itemize}



}
%-----------------------------------------------------------------------------------
\frame{
\frametitle{Modelo en el AFM}
A diferencia del ACP, el AFM considera un modelo estadístico de describe las variable de estudio, este es:
\begin{equation}
\boxed{\mathbf{X}=\Lambda \mathbf{F}+\mathbf{\epsilon}}
\end{equation}
donde:
\begin{itemize}
\item $\mathbf{X}$ es la matriz de variables observadas.
\item $\Lambda$ es la matriz de cargas factoriales.
\item $\mathbf{F}$ es el vector de factores.
\item $\mathbf{\epsilon}$ es el vector de errores.
\end{itemize}
\begin{align*}
X_{1}&=\lambda_{11}f_{1}+\lambda_{12}f_{2}+\ldots+\lambda_{p1}f_{k}+\epsilon_{1}\\
&\vdots\\
X_{p}&=\lambda_{p1}f_{1}+\lambda_{p2}f_{2}+\ldots+\lambda_{pk}f_{k}+\epsilon_{p}
\end{align*}

}
